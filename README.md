# Aviation Calculation API
A collection of functions related to air navigation and aviation. This API is based on Aviation Formulary 1.46 by Ed Williams and is written in C and then ported to VBA. The original use of this API was to do navigational calculations within Microsoft Excel when working with an external .dll file is not an option.

The original work by Ed Williams can be found at https://edwilliams.org/ with the latest release at https://edwilliams.org/avform147.htm
